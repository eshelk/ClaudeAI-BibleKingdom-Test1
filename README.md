# 말씀나라 — 주일학교 플랫폼

주일학교 학생들을 위한 픽셀 아트 스타일의 성경 게임 플랫폼입니다.
게임을 통해 말씀을 배우고, 달란트를 모아 천국 집을 꾸며나갈 수 있습니다.

## 페이지 구성

| 파일 | 설명 |
|------|------|
| `index.html` | 메인 랜딩 페이지 |
| `game-center.html` | 게임 센터 (말씀 퀴즈, 미션 등) |
| `ranking.html` | 학생 랭킹 페이지 |
| `teacher-dashboard.html` | 교사용 관리 대시보드 |

## 기술 스택

- 순수 HTML / CSS / JavaScript (프레임워크 없음)
- Google Fonts: Noto Sans KR, Press Start 2P (픽셀 폰트)
- 인라인 스타일 & 스크립트 (단일 파일 구성)

## 실행 방법

별도의 빌드 과정 없이 `index.html`을 브라우저로 바로 열면 됩니다.
